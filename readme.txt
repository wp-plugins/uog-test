MCQ type online test plugin.


== Description ==

This plugin is used to take online MCQ type test, check answers realtime and display result at the end of the result.


= Features =
* Add new questions/answers
* MCQ type test (work with any theme)
* Realtime answer checking
* Save result in database.

== Installation ==

1. Upload the plugin to your 'wp-content/plugins' directory
1. Activate the plugin
